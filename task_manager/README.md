# 📃 タスク管理メモ帳 (Task Manager CLI)

簡単な CLI (コマンドライン)型のタスク管理ツールです。

---

## ✨ 作れること

* タスクの追加
* タスク一覧の表示
* タスクの削除
* タスクの修正
* タスクは `tasks.txt`に保存され、次回起動時も続きます

---

## 💻 使用方法

### 1. 実行

```bash
python task_manager.py
```

### 2. メニュー画面

```
=== タスク管理メモ帳 ===
1. タスクを追加
2. タスク一覧を表示
3. タスクを削除
4. 終了
5. タスクを修正
```

---

## 🔧 動作環境

* Python 3.x
* CLI 環境 (例: VSCode ターミナル, Mac/Linuxの純粋ターミナル)
